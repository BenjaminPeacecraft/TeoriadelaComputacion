## Introducción y Objetivo
El propósito de esta práctica es profundizar en la comprensión y aplicación de los Autómatas Finitos No Deterministas (AFND) mediante el uso de JFLAP. Adicionalmente, se desarrolló una aplicación con interfaz gráfica en Java (Swing) que permite simular Autómatas Finitos Deterministas (AFD) a partir de diferentes formatos de entrada nativos e intercambiables.

## Instrucciones de Instalación y Ejecución
La aplicación de simulación fue desarrollada en Java utilizando la biblioteca estándar Swing para la interfaz gráfica y el analizador XML nativo de Java para leer los archivos .jff generados por JFLAP.

**Requisitos:**
* Java Development Kit (JDK) 8 o superior instalado.
* Archivos .jff o .xml válidos generados desde JFLAP.

**Pasos para ejecutar:**
1. Clona este repositorio en tu máquina local.
2. Compila el código fuente desde la terminal con el comando: javac SimuladorAFD.java
3. Ejecuta la aplicación con el comando: java SimuladorAFD

** Ejemplos **
Se incluyen 3 automatas que se pueden cargar en el preograma con el fin de observar el funcionamiento de este.
